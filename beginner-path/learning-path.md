# Beginner AI Learning Path

## Level 1: Foundations

Learn the language of modern AI.

- AI: software that performs tasks normally requiring human intelligence
- LLM: a large language model that predicts and generates text
- Prompt: the instruction you give the model
- Context window: the information the model can use at one time
- Hallucination: a confident answer that may be wrong
- Tool use: when an AI system calls an external tool, API, file, or application

## Level 2: Practical Prompting

Practice prompts that ask the model to:

- Explain a topic simply
- Compare options
- Create a checklist
- Summarize a document
- Draft an email
- Build a study plan
- Identify risks

Use this structure:

```text
Role: You are a helpful expert in [topic].
Goal: Help me [specific goal].
Context: Here is what you need to know: [details].
Output: Format the answer as [table/checklist/steps].
Constraints: Keep it [short/practical/beginner-friendly].
```

## Level 3: Workflow Thinking

A workflow is a repeatable process.

Example:

1. Collect input
2. Clean or organize it
3. Ask AI to analyze it
4. Review the output
5. Save the result
6. Improve the prompt

## Level 4: Safety Basics

Beginner safety rules:

- Do not paste passwords, secrets, API keys, or private health information into public AI tools.
- Verify important claims using trusted sources.
- Ask the model to show uncertainty.
- Use human review for medical, legal, financial, and security decisions.
- Treat AI output as a draft, not the final truth.

## Level 5: Portfolio Projects

Build small projects and document:

- What problem it solves
- Who it helps
- How it works
- What tools are used
- What risks exist
- How a human reviews the output
