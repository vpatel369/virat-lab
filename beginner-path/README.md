# Beginner Path

This section is for people with zero or limited technical background who want to become AI-ready through practical learning.

## Goal

Help anyone build a working foundation in AI within 90 days by learning the basics, practicing prompts, building small workflows, and understanding safety.

## What You Will Learn

1. Computer and internet basics
2. Cloud and API fundamentals
3. AI and LLM fundamentals
4. Prompt engineering
5. Automation basics
6. AI tools for research and productivity
7. AI security and privacy basics
8. Simple portfolio projects

## 90-Day Learning Path

| Phase | Days | Focus | Outcome |
| --- | --- | --- | --- |
| 1 | 1-15 | Digital foundations | Understand files, browsers, accounts, cloud, and basic security |
| 2 | 16-30 | AI fundamentals | Explain LLMs, prompts, tokens, hallucinations, and limitations |
| 3 | 31-45 | Prompt engineering | Build reusable prompts for work, research, and learning |
| 4 | 46-60 | Automation | Design simple workflows using AI tools, APIs, or no-code platforms |
| 5 | 61-75 | Security and privacy | Identify sensitive data, prompt injection, and unsafe outputs |
| 6 | 76-90 | Portfolio | Publish 2-3 small projects with clear documentation |

## Beginner Projects

- Personal AI learning journal
- Prompt library for daily tasks
- Research summary workflow
- Resume improvement assistant prompt
- Secure chatbot checklist
- Simple automation idea map

## Key Mindset

You do not need to know everything before you start. Learn one concept, build one small thing, and document what you learned.
