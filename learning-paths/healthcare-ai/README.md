# Healthcare AI Learning Path

This path teaches safe, practical healthcare AI workflows for research, administration, education, and clinical support contexts.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Healthcare AI concepts, privacy, safety, and human review |
| Practice | Research summaries, patient education drafts, and workflow risk checks |
| Build | Compliance-aware healthcare AI workflow |
| Share | Portfolio project with privacy and safety notes |

## Levels

### Beginner

Understand healthcare AI use cases, PHI/PII awareness, and AI limitations.

### Intermediate

Design safe research and administrative workflows with review gates.

### Advanced

Evaluate healthcare AI systems for compliance, bias, governance, observability, and clinical risk.
