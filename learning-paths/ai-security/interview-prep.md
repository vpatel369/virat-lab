# Interview Prep

## Beginner Questions

- What is prompt injection?
- Why is sensitive data disclosure a risk in LLM apps?
- What is insecure output handling?
- Why is human review important?

## Intermediate Questions

- How do you threat model a RAG application?
- How would you secure tools used by an AI agent?
- What logs are useful for investigating an AI incident?
- How do OWASP LLM risks map to enterprise controls?

## Advanced Questions

- How would you design runtime governance for autonomous agents?
- How do you secure MCP servers and connectors?
- How do you prevent identity abuse by nonhuman agents?
- How would you build an AI red-team program?

An internal HR chatbot uses RAG over policy documents and can open tickets in ServiceNow.
