## Prompt Injection Exercise

1. Create a safe test assistant prompt.
2. Add untrusted user content that asks the assistant to ignore instructions.
3. Record whether the model follows trusted or untrusted instructions.
4. Add mitigation: content labeling, refusal rules, output validation, and tool restrictions.

## MCP Security Exercise

- List every MCP server or tool.
- Identify what data it can read.
- Identify what actions it can perform.
- Add allowlists, authentication, logging, and approval gates.
