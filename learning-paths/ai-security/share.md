# Share

## GitHub Guidance

Your AI security portfolio should show judgment. Do not publish exploit code against systems you do not own.

Include:

- Safe lab scope
- Threat model
- Screenshots or sample results
- Mitigations
- Lessons learned
- Ethical testing statement

## LinkedIn Post Template

```text
I completed an AI security lab focused on [prompt injection/RAG/agents].

I practiced:
- threat modeling
- safe red-team testing
- governance controls
- observability planning

Key lesson: [lesson]
GitHub: [link]
```
