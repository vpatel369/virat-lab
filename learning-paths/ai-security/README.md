# AI Security Learning Path

This path teaches learners how to secure LLM applications, RAG systems, AI agents, automation workflows, and enterprise AI programs.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | AI security vocabulary, risk models, and governance concepts |
| Practice | Prompt injection, threat modeling, red-team, and observability labs |
| Build | AI security portfolio with labs, checklists, and architecture reviews |
| Share | GitHub project, write-up, demo walkthrough, and interview stories |

## Levels

### Beginner

Understand LLM risks, prompt injection, sensitive data leakage, and human review.

### Intermediate

Threat model RAG apps, agent workflows, tool use, APIs, identity, logging, and deployment patterns.

### Advanced

Design runtime governance, AI red-team plans, MCP security controls, observability, and enterprise AI governance programs.
