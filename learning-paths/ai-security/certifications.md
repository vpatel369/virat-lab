# Certifications

AI security is still emerging, so combine security credentials with AI-specific projects.

## Useful Certification Areas

- Security fundamentals
- Cloud security
- Identity and access management
- Security operations
- Governance, risk, and compliance
- AI fundamentals

## Resources

- [Microsoft Security training](https://learn.microsoft.com/en-us/security/)
- [AWS Security Learning Plan](https://explore.skillbuilder.aws/learn/public/learning_plan/view/91/security-learning-plan)
- [Microsoft Identity training](https://learn.microsoft.com/training/identity-and-access/)
- [SANS AI resources](https://www.sans.org/artificial-intelligence/)

## Strategy

Pair every certification with a project. Example: if studying cloud security, build an AI agent permission review for cloud APIs.
