# Claude Skills Learning Path

This path teaches how to design reusable Claude Skills for coding, research, security, automation, and documentation workflows.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Skill anatomy and workflow design |
| Practice | Skill reviews, templates, and examples |
| Build | Reusable skill package |
| Share | Skill README, demo, and usage guide |

## Levels

### Beginner

Understand what a skill is and when it should be used.

### Intermediate

Create structured skill instructions with templates, references, and examples.

### Advanced

Build multi-file skills for AI security, automation, research, and observability workflows.
