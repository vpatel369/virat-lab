# Learn

## Concepts

- Skill trigger: when the assistant should use the skill.
- Workflow: steps the assistant should follow.
- References: supporting docs, examples, templates, or scripts.
- Output contract: the final artifact the skill should produce.
- Safety boundaries: what the skill should avoid.
