# Interview Prep

## Beginner

- What is a Claude Skill?
- What makes a skill reusable?
- What should a skill include?

## Intermediate

- How would you test a skill?
- How do templates improve reliability?
- How do you decide when not to use a skill?

## Advanced

- How would you design a skill for AI security review?
- How would you package references and examples?
- How do you prevent unsafe skill behavior?
