# Learning Paths

Virat Lab learning paths follow one repeatable model:

```text
LEARN -> PRACTICE -> BUILD -> SHARE
```

This keeps the repository from becoming a random link collection. Every path helps learners understand concepts, practice hands-on, build portfolio artifacts, and share evidence of skill.

## Paths

| Path | Best For | Primary Outcome |
| --- | --- | --- |
| [beginner](beginner/) | New AI learners | AI-ready foundation and first portfolio project |
| [ai-security](ai-security/) | Security, cloud, IAM, and AI practitioners | AI security lab portfolio |
| [prompt-engineering](prompt-engineering/) | Knowledge workers, builders, analysts | Tested prompt library and evaluation habit |
| [claude-skills](claude-skills/) | Claude Code and workflow builders | Reusable skill packages |
| [ai-agents](ai-agents/) | Agent builders and automation engineers | Governed agent workflow |
| [automation](automation/) | Operations and workflow builders | AI automation system with review gates |
| [cloud-identity-security](cloud-identity-security/) | IAM, cloud, and security professionals | AI identity security review toolkit |
| [healthcare-ai](healthcare-ai/) | Healthcare, research, and compliance learners | Safe healthcare AI workflow design |
