# Cloud + Identity Security Learning Path

This path connects AI systems with cloud security, IAM, nonhuman identities, agent permissions, and enterprise governance.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Cloud, IAM, Zero Trust, and AI identity concepts |
| Practice | Permission reviews, access mapping, and policy design |
| Build | AI identity security review toolkit |
| Share | Architecture review, checklist, and interview stories |

## Levels

### Beginner

Learn authentication, authorization, RBAC, MFA, and least privilege.

### Intermediate

Review cloud identities, service accounts, OAuth apps, API keys, and AI tool permissions.

### Advanced

Design identity governance for AI agents and nonhuman workloads.
