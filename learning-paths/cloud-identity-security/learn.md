# Learn

## Concepts

- Authentication: proving identity.
- Authorization: deciding what an identity can access.
- RBAC: role-based access control.
- Least privilege: minimum access needed.
- Nonhuman identity: service accounts, apps, agents, workloads, and automation identities.
- Agent identity: how an AI agent is registered, scoped, monitored, and revoked.
