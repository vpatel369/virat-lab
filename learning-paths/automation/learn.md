# Learn

## Concepts

- Trigger: event that starts a workflow.
- Input: data entering the workflow.
- AI task: summarization, extraction, classification, drafting, or routing.
- Human review: approval before high-impact action.
- Output: document, ticket, alert, message, or database update.
- Monitoring: logs, errors, alerts, and performance metrics.
