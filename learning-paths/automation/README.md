# AI Automation Learning Path

This path teaches how to turn repeated work into AI-assisted workflows with clear triggers, review gates, logs, and safety controls.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Workflow, API, webhook, and automation patterns |
| Practice | n8n, GitHub Actions, prompt routing, and review tasks |
| Build | AI automation system |
| Share | Workflow diagram, demo, and operational checklist |

## Levels

### Beginner

Map manual tasks and identify safe automation candidates.

### Intermediate

Use n8n, APIs, webhooks, structured outputs, and human approvals.

### Advanced

Build monitored, secure, auditable AI workflow systems.
