# Learn

## Core Concepts

- Role: the perspective the model should use.
- Task: the action the model should perform.
- Context: information the model needs.
- Constraints: rules, boundaries, and tone.
- Output format: table, checklist, JSON, markdown, or step-by-step response.
- Evaluation: testing whether a prompt works consistently.
