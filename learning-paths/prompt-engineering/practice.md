# Practice

## Exercises

| Exercise | Goal |
| --- | --- |
| Rewrite a vague prompt | Improve specificity |
| Compare outputs | Test two prompt versions |
| Add constraints | Reduce risky or irrelevant output |
| Create a rubric | Evaluate answer quality |
| Add examples | Improve consistency |

## Hands-On Prompt Test

1. Write a baseline prompt.
2. Run it on three sample inputs.
3. Score outputs for clarity, correctness, usefulness, and safety.
4. Rewrite the prompt.
5. Compare results.

## Progress Checklist

- [ ] I can turn a vague request into a structured prompt.
- [ ] I can ask for tables, checklists, and JSON-style outputs.
- [ ] I can write a simple evaluation rubric.
