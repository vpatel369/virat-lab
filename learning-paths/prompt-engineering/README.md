# Prompt Engineering Learning Path

Prompt engineering is the practice of designing clear, testable instructions that produce useful, reliable AI outputs.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Prompt patterns, context design, structured outputs, and evaluation |
| Practice | Prompt rewrites, comparison tests, and eval rubrics |
| Build | Prompt library with examples and quality checks |
| Share | GitHub prompt pack, article, and demo walkthrough |

## Levels

### Beginner

Write prompts with role, task, context, constraints, and output format.

### Intermediate

Use few-shot examples, structured outputs, prompt versioning, and evaluation.

### Advanced

Design prompt systems for workflows, agents, governance, security testing, and production reliability.
