# Certifications

Prompt engineering is best proven through artifacts.

## Useful Learning

- OpenAI and Anthropic official docs
- DeepLearning.AI short courses
- Hugging Face learning resources
- Domain-specific certifications in security, cloud, healthcare, or analytics

## Credential Strategy

Combine prompt engineering with a domain. A prompt engineer who understands security, healthcare, automation, or research is more useful than a generic prompt writer.
