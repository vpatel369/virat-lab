# AI Agents Learning Path

This path teaches how to design, build, observe, and secure AI agents that use tools, memory, retrieval, handoffs, and approvals.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Agent architecture and risk vocabulary |
| Practice | Tool-use, guardrail, tracing, and approval exercises |
| Build | Governed agent workflow |
| Share | Demo, architecture diagram, and security review |

## Levels

### Beginner

Understand goals, tools, memory, and human review.

### Intermediate

Build simple workflows using LangChain, LangGraph, CrewAI, n8n, or OpenAI Agents SDK concepts.

### Advanced

Design agent governance: identity, policy enforcement, MCP security, tool guardrails, tracing, and kill switches.
