# Learn

## Concepts

- Agent: an AI system that can plan and take actions using tools.
- Tool: an API, function, file operation, browser action, database query, or external integration.
- Memory: stored information used across steps or sessions.
- Handoff: passing work between agents or specialized roles.
- Guardrail: validation that blocks or shapes unsafe input, output, or tool calls.
- Trace: a record of model calls, tool calls, handoffs, and guardrail events.
