# Interview Prep

## Beginner

- What is an AI agent?
- What is a tool call?
- Why does an agent need human review?

## Intermediate

- How would you design a safe tool permission model?
- What should be logged during an agent run?
- How do you handle tool failure?

## Advanced

- How would you prevent rogue agents?
- How would you design identity for nonhuman agents?
- How do guardrails differ from runtime policy enforcement?
- How would you secure MCP servers?
