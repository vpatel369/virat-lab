# Practice

## Exercises

| Exercise | Goal |
| --- | --- |
| Tool inventory | List every action an agent can take |
| Permission review | Restrict tools to least privilege |
| Guardrail design | Define input, output, and tool-call checks |
| Trace review | Inspect model calls, tool calls, and handoffs |
| Failure simulation | Test bad input, tool failure, and unsafe requests |

## Hands-On Flow

1. Design a research assistant that can summarize documents.
2. Add one tool at a time.
3. Document what each tool can read and change.
4. Add a human approval step before external actions.
5. Log every tool call.
