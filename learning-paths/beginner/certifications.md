# Certifications

Certifications are optional at this stage. Use them to structure learning, not to replace projects.

## Beginner-Friendly Options

- Microsoft Learn AI fundamentals modules
- Google AI learning resources
- DeepLearning.AI short courses
- Hugging Face learning paths

## How To Choose

- Pick a certification only if it matches your goal.
- Build one project while studying.
- Document what you learned in GitHub.

## Career Note

For entry-level learners, a clear portfolio often communicates practical ability better than collecting many certificates.
