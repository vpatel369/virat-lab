# Roadmap

| Week | Focus | Outcome |
| --- | --- | --- |
| 1 | AI vocabulary | Explain core concepts |
| 2 | Prompt basics | Create reusable prompts |
| 3 | Research skills | Summarize and verify sources |
| 4 | Workflow thinking | Map repeatable tasks |

| 6 | Build | Publish first project |

## Level Transitions

- Beginner to Intermediate: you can write prompts with role, task, context, constraints, and output format.
- Intermediate to Advanced: you can design a workflow, test it, document limitations, and explain risks.
