# Build

## Portfolio Project: AI Learning Assistant

Build a small GitHub project that helps a beginner learn any topic.

## Requirements

- `README.md` explaining the problem and audience
- 10 reusable prompts
- A safety checklist
- One example output
- A reflection on what worked and what failed

## Step-By-Step

1. Pick a topic such as cybersecurity, healthcare AI, or automation.
2. Create prompts for explaining, quizzing, summarizing, and planning.
3. Test each prompt with sample content.
4. Improve unclear prompts.
5. Document limitations and review steps.

## Advanced Extension

Add a simple scoring rubric to evaluate whether answers are clear, accurate, and useful.
