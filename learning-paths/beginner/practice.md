# Practice

## Daily Exercises

| Exercise | Prompt |
| --- | --- |
| Explain | `Explain [topic] to a beginner with one example and one mistake to avoid.` |
| Summarize | `Summarize this text into key points, risks, and next actions.` |
| Compare | `Compare these options in a table with pros, cons, and recommendation.` |
| Plan | `Create a 7-day study plan for [topic] with daily outputs.` |

## Hands-On Tasks

- Create a personal prompt notebook.
- Summarize one article using three different prompts.
- Ask an AI tool to produce a checklist, then improve it manually.
- Use a fake sample dataset to practice safe data handling.

## Progress Checklist

- [ ] I can explain AI basics without jargon.
- [ ] I can write a structured prompt.
- [ ] I can check an AI answer before trusting it.
- [ ] I can identify private data that should not be pasted into public tools.
