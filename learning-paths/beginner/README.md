# Beginner Learning Path

This path helps a new learner become AI-ready by moving from basic concepts to practical projects.

## LEARN -> PRACTICE -> BUILD -> SHARE

| Stage | Output |
| --- | --- |
| Learn | Plain-English AI vocabulary and tool awareness |
| Practice | Daily prompts, safe research, and workflow exercises |
| Build | First AI learning portfolio project |
| Share | GitHub README, LinkedIn post, and demo notes |

## Levels

### Beginner

Learn AI basics, prompt structure, privacy awareness, and how to ask better questions.

### Intermediate

Use AI for research, automation planning, and structured outputs.

### Advanced

Compare tools, design repeatable workflows, and explain AI limitations to others.

## Transition Guidance

Move to [prompt-engineering](../prompt-engineering/) when you can write clear prompts. Move to [ai-security](../ai-security/) when you can explain why sensitive data and tool access matter.
