# Start Here - Virat Lab

Welcome to Virat Lab.

This repository helps beginners, professionals, healthcare workers, cybersecurity engineers, automation builders, and future AI practitioners learn practical AI skills.

## What Is Virat Lab?

Virat Lab is an open AI learning ecosystem focused on:

- AI fundamentals
- Prompt engineering
- AI security
- Healthcare AI
- Automation systems
- Agentic AI
- Claude Skills
- Research workflows
- Future-ready career skills

The goal is practical education: learn the concepts, build small projects, understand the risks, and document reusable workflows.

## Beginner Learning Path

1. Learn what AI, LLMs, prompts, tokens, and tools are.
2. Practice daily prompts for writing, summarizing, researching, and planning.
3. Build a simple automation workflow.
4. Learn basic AI security risks like prompt injection and data leakage.
5. Create one portfolio project using a template from [templates/](templates/).

## Practical Rule

Do not only read. Build something small every week.

Examples:

- A prompt library for your job
- A research summary workflow
- A secure chatbot checklist
- A healthcare note summarization workflow with privacy safeguards
- A simple n8n or GitHub automation
- A Claude Skill template
