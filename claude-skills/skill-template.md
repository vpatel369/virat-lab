# Claude Skill Template

Use this template when designing a new skill.

```markdown
# Skill Name

## Purpose

Explain what this skill helps with.

## Use When

- Scenario 1
- Scenario 2
- Scenario 3

## Do Not Use When

- Scenario where another workflow is better
- Scenario that needs human expert review

## Inputs

- Required input 1
- Required input 2
- Optional input 1

## Workflow

1. Understand the user's goal.
2. Gather the minimum needed context.
3. Follow the specialized steps.
4. Produce the requested output.
5. Verify quality and safety.

## Output Format

Describe the final answer, file, checklist, or artifact.

## Safety Notes

- Do not expose secrets.
- Flag medical, legal, financial, and security uncertainty.
- Ask for approval before high-risk actions.
```
