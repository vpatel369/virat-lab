# Claude Skills

This section documents Claude Skills, AI coding workflows, prompt chains, automation patterns, and reusable AI engineering techniques.

## What Are Claude Skills?

Claude Skills are structured instructions and supporting files that help an AI assistant perform a specialized task consistently. A good skill explains when to use it, what steps to follow, what files or templates matter, and how to produce a reliable output.

## What This Section Includes

- AI coding workflows
- Agentic systems
- Prompt chains
- Automation templates
- Research agents
- AI copilots
- Skill design patterns
- Skill templates

## Learning Goals

- Understand what makes a skill reusable
- Create clear skill instructions
- Package templates and references
- Build practical workflows for coding, research, security, and automation
- Avoid vague prompts by documenting repeatable process
