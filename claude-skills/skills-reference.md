# Claude Skills Reference

## Anatomy Of A Good Skill

A strong skill usually includes:

- Name and purpose
- When to use it
- When not to use it
- Step-by-step workflow
- Required inputs
- Output format
- Templates or examples
- Safety notes

## Quality Checklist

- [ ] Clear trigger: when should the skill be used?
- [ ] Clear workflow: what steps should happen?
- [ ] Clear output: what should be produced?
- [ ] Examples included
- [ ] Safety limits included
- [ ] Beginner-friendly language
- [ ] Advanced notes where useful
