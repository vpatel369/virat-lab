# Workflows

This section contains repeatable workflows for AI agents, GitHub, n8n, research, security, automation, and documentation.

## What Makes A Good Workflow?

A good workflow has:

- Clear trigger
- Clear inputs
- Clear steps
- Clear output
- Error handling
- Human review where needed
- Security and privacy notes
- Reusable documentation

## Covered Areas

- AI agents
- GitHub workflows
- n8n automation
- AI automation systems
- Research workflows
- Security review workflows
- Documentation workflows
