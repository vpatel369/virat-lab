# Contributing To Virat Lab

Thank you for helping improve Virat Lab.

## Contribution Principles

- Keep content practical.
- Write for beginners first, then add advanced notes.
- Preserve existing useful content.
- Prefer clear markdown structure.
- Add examples, checklists, and templates.
- Include security and privacy notes when relevant.

## Good Contributions

- New learning paths
- Hands-on labs
- Prompt libraries
- Workflow templates
- AI security checklists
- Healthcare AI safety notes
- Research summaries
- Tool comparisons

## Style Guide

- Use clear headings.
- Use tables for comparison.
- Use checklists for actions.
- Avoid hype.
- Explain acronyms.
- Link related files.
- Keep examples safe and ethical.
