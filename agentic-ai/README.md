# Agentic AI

Agentic AI systems can plan, use tools, call APIs, retrieve information, write files, and complete multi-step tasks with varying levels of autonomy.

## Why This Matters

Agents are powerful because they can move beyond single-turn chat. They are also risky because they can take actions across real systems.

## Core Concepts

- Goals: what the agent is trying to accomplish
- Planning: how the agent breaks work into steps
- Tools: APIs, browsers, files, databases, and applications the agent can use
- Memory: information saved across steps or sessions
- Orchestration: how multiple tools or agents coordinate
- Guardrails: limits, validation, and approvals
- Observability: logs, traces, and review points

## Learning Order

1. Understand tool use
2. Build a simple agent workflow
3. Add logging and human review
4. Limit permissions
5. Test failure modes
6. Document the workflow
