- Prompt structure
- Context design
- Structured outputs
- Evaluation rubrics
- Domain expertise
- Safety and privacy awareness

4. Share before/after prompt improvements.

## Salary And Hiring Trends

Standalone prompt engineer roles are less stable than domain roles that require prompt engineering. Strong positioning: "security analyst with prompt engineering," "healthcare researcher with AI workflows," or "automation builder with prompt evaluation."

## Interview Prep

- How do you test prompts?
- How do you reduce hallucinations?
- How do you design prompts for high-risk domains?
