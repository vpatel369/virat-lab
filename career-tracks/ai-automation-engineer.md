# AI Automation Engineer

## Required Skills

- Workflow mapping
- APIs and webhooks
- n8n or similar automation tools
- Prompt design
- Error handling
- Human approval gates
- Security and privacy basics

## Roadmap

1. Complete [Automation](../learning-paths/automation/).
2. Complete [Prompt Engineering](../learning-paths/prompt-engineering/).
3. Build one workflow with logs and review.
4. Document before/after process value.

## Projects

- Support ticket triage workflow
- Meeting summary automation
- Security alert enrichment
- Weekly research digest automation

## Certifications

Cloud fundamentals, API training, project management, and security fundamentals are useful.

These roles often overlap with software developer, automation specialist, business systems, RevOps, and solutions engineering roles. Portfolio evidence is especially important because job titles are still evolving.

- How do you choose what to automate?
- How do you prevent unsafe AI actions?
