# AI Security Learning Path

## Beginner Level

Start here if you are new to AI security.

### Learn

- What AI applications are
- What LLMs can and cannot do
- What prompt injection means
- Why sensitive data handling matters
- Why human review is important

### Build

- A list of unsafe prompts
- A secure chatbot checklist
- A one-page AI risk summary

## Intermediate Level

### Learn

- OWASP Top 10 for LLM Applications
- RAG security
- API security
- Identity and access management
- AI logging and monitoring
- Secure deployment patterns

### Build

- Prompt injection lab
- RAG data review checklist
- AI threat model
- Security test plan

## Advanced Level

### Learn

- Agentic AI security
- Multi-agent workflow risk
- AI red teaming
- Model supply chain risk
- Governance at enterprise scale
- AI incident response

### Build

- Agent permission review
- AI red team report
- AI SOC workflow
- Enterprise AI governance package

## Skill Matrix

| Skill | Beginner | Intermediate | Advanced |
| --- | --- | --- | --- |
| Prompt injection | Recognize examples | Test simple apps | Design mitigations |
| RAG security | Understand retrieval | Review data sources | Test poisoning paths |
| Agent security | Understand tool use | Review permissions | Design approval gates |
| Governance | Use checklists | Map controls | Build operating model |
| Red teaming | Run examples | Write test cases | Lead simulations |
