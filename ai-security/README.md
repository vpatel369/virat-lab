# AI Security

This section focuses on practical AI security for modern enterprises, security teams, builders, and future-ready professionals.

## Why AI Security Matters

AI systems are now connected to documents, databases, browsers, APIs, identity systems, code repositories, and business workflows. That creates new risks:

- Users can manipulate prompts.
- Models can leak sensitive information.
- Agents can take unsafe actions.
- Retrieval systems can ingest malicious content.
- Automation can scale mistakes quickly.
- Governance can fall behind adoption.

AI security is the practice of designing, testing, monitoring, and governing these systems so they are useful and trustworthy.
