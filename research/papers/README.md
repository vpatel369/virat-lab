# Papers

Use this folder for paper summaries, research notes, and practical translation.

## Paper Review Flow

1. Read abstract and conclusion.
2. Identify the problem.
3. Identify method and evidence.
4. List limitations.
5. Translate into practical workflows.
6. Add security, privacy, and governance notes.

## Suggested Sources

- [arXiv](https://arxiv.org/)
- [Semantic Scholar](https://www.semanticscholar.org/)
- [PubMed](https://pubmed.ncbi.nlm.nih.gov/)
- [Papers with Code](https://paperswithcode.com/)

## Output Standard

Every review should include:

- Plain-English summary
- Practical use cases
- Risks and limitations
- Questions for follow-up
