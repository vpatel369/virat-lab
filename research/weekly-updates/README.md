# Weekly Updates

Use this folder to publish weekly AI learning digests.

## Format

```markdown
# Weekly AI Update - YYYY-MM-DD

## Theme

## Top Developments

## Practical Lessons

## Security Notes

## Tools To Test

## Papers Or Reports

## Career Takeaways

## Next Week Questions
```

## Workflow

1. Pick one theme.
2. Collect primary sources.
3. Summarize practical impact.
4. Add risks and open questions.
5. Share one concise post publicly.
