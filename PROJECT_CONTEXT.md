# Virat Lab - Project Context

## Vision

Virat Lab is an open-source AI learning ecosystem focused on helping beginners and professionals become future-ready in the AI era.

The project combines:

- AI Security
- Agentic AI
- Claude Skills
- Prompt Engineering
- Automation
- Healthcare AI
- Cloud and Identity Security
- Research workflows
- AI education

## Primary Goals

1. Create beginner-friendly AI learning paths
2. Build practical AI security education
3. Curate real-world AI workflows
4. Create reusable Claude Skills systems
5. Build scalable AI automation workflows
6. Develop future-ready career resources
7. Turn the GitHub repo into a learning platform
8. Structure every major topic around LEARN -> PRACTICE -> BUILD -> SHARE

The repo should always be:
