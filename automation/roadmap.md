# Automation Roadmap

## Phase 1: Process Thinking

- Identify repeated work
- Define inputs and outputs
- Write the manual process
- Identify decision points
- Identify failure points

## Phase 2: No-Code Workflows

- Build simple trigger-action workflows
- Add AI summarization or classification
- Save outputs to a document, table, or ticket
- Test with safe sample data

## Phase 3: API Workflows

- Learn API basics
- Use webhooks
- Parse JSON
- Handle errors
- Store secrets safely

## Phase 4: AI-Assisted Automation

- Prompt templates
- Structured outputs
- Human review
- Evaluation examples
- Logging and monitoring

## Phase 5: Production Automation

- Access controls
- Retry logic
- Audit logs
- Rollback plans
- Security review
- Documentation
